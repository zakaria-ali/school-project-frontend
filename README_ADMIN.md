# دليل استخدام لوحة تحكم معرض الصور

## كيفية الوصول:

1. أنشئ ملف `.env` من `ENV_TEMPLATE.txt`
2. أضف كلمة المرور في `.env`
3. افتح: `http://localhost:5173/admin/gallery`
4. أدخل كلمة المرور من `.env`

## تغيير كلمة المرور:

افتح ملف `.env` وغيّر:
```env
VITE_ADMIN_PASSWORD=كلمة_مرور_جديدة_قوية
ADMIN_PASSWORD=كلمة_مرور_جديدة_قوية
```

## تشغيل النظام:

### الطريقة الأولى: مع Backend (موصى به)

1. **ثبت المكتبات المطلوبة:**
   ```bash
   npm install
   ```

2. **شغّل السيرفر والموقع معاً:**
   ```bash
   npm run dev:all
   ```

   أو شغّلهما منفصلين:
   ```bash
   # Terminal 1: السيرفر
   npm run dev:server

   # Terminal 2: الموقع
   npm run dev
   ```

### الطريقة الثانية: بدون Backend (localStorage فقط)

1. **شغّل الموقع فقط:**
   ```bash
   npm run dev
   ```

2. **الصور ستُحفظ في localStorage**
   - ⚠️ ملاحظة: الصور ستُحفظ كـ Base64 (قد تكون كبيرة)
   - ⚠️ الصور قد تُفقد عند مسح بيانات المتصفح

## المميزات:

✅ رفع صور مباشرة من المتصفح
✅ إضافة عنوان وتصنيف لكل صورة
✅ معاينة الصورة قبل الرفع
✅ حذف الصور
✅ عرض جميع الصور الموجودة
✅ تحديث تلقائي لصفحة المعرض

## هيكل الملفات:

```
public/
  images/
    gallery/          # الصور المرفوعة (يُنشأ تلقائياً)
  api/
    gallery.json      # بيانات الصور (يُحدث تلقائياً)
```

## نصائح:

1. **لرفع صور كبيرة:** استخدم Backend (الطريقة الأولى)
2. **لحل سريع:** استخدم localStorage (الطريقة الثانية)
3. **للتأكد من الحفظ:** تحقق من ملف `public/api/gallery.json`
4. **لحذف جميع الصور:** امسح localStorage أو ملف JSON
